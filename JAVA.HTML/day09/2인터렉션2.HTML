배열의 순환할 수 있는 구조를 이용하여 특정 기능을 할 수 있는 메서드가 제공됨

  • array.reduce(callback function(total,value,index,self),[init value]) : 내부 연산의 누적값을 리턴
  • array.reduceRight(reduce와 동일) : 배열의 값을 우측에서 좌측 방향으로 대입
  • array.every(callback function(value,index,self)) : 콜백 함수의 리턴 조건에 모든 배열의 값이 만족하면 리턴 아니면 펄스 리턴
  • array.some(callback function(value,index,self)) : 콜백 함수의 리턴 조건에 만족하는 배열값이 존재하면 투르리턴 아니면 펄스
  • array.indexOf("str",[start index])
  • array.lastIndexOf("str",[start index])
  • array.find(callback function(value,index,self))
  • array.findIndex(callback function(value,index,self))
  • array.from()
