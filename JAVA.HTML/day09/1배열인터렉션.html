배열의 순환 할수 있는 구조를 이용하여 특정 기능을 할수 있는 메서드가 제공됨

  • array.forEach(callback function) : 일반 반복문과 유사한 형태로 배열 탐색 array.forEach(function(element_value, index, array)){ code; }
  • array.map(callback function) : forEach를 수행하면서 연산결과를 리턴하여 새로운 배열을 만들어줌
  • array.filter(callback function) : map처럼 수행 + 비교연산의 결과에서 true에 해당하는 값만 필터링 하여 리턴함
