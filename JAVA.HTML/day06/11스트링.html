가장 많이 사용하는 데이터 타입을 다루는 메서드

  • 문자열 타입은 인덱스가 0부터 시작
  • 문자열 데이터 값 내에서는 개행을 하는것은 추천하는 방법이 x
  • length 프로퍼티는 문자값의 총길이, 마지막 문자 번호는 length-1
  • 문자열 추출 : silce(), substring(), substr()
  • replace(oldstr,newstr) : 첫번째 매칭되는 문자만, 대소문자 구분
  • padstart(목표문자길이,채울문자) : 앞쪽에 채우기 padEnd(목표문자길이,채울문자) : 뒤쪽에 채우기
  • charAt(인덱스번호) : 해당 위치의 문자만 추출
  • indexof("str"), lastindexof("str") : 지정 문자열의 인덱스 리턴
  • Startwith(),endwith() : 지정문자열이 탐색해당 문자열에서 시작 과 끝에 존재하는지 여부를 체크 return true or false
  • Startwith(), endwith() : 지정 문자열이 탐색대상 문자열에서 시작과 끝에 존재하는지
