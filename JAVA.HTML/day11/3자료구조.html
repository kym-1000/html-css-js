자바스크립트에서 데이터 타입 확장 개념으로 집합데이터를 다룰수 있는 객체를 만듬

  • set 안의 중복데이터는 허용하지 않음 : 중복 검사 후 존재하는 데이터의 최초의 데이터만 저장
  • set() : 파라미터로는 주로 배열을 사용한다 => 배열의 인덱스와 값을 활용하여 집합 데이터를 만듬
  • set의 전용 메서드 들이 존재
  • add : 값을 추가 : setName.add(값)
  • size : 크기를 알려줌 : setName.size; size()이 아님
  • has : 값이 있는지 체크 : setName.has(값) 있으면 트루 없으면 펄스
  • delete : 값을 삭제 : setName.delete(값) 성공하면 트루 없으면 펄스 반환
  • clear : 모두 삭제 setName.clear()
  • forEach(v,i,self(set)) : 인덱스 = 밸류 (set은 키가 별도로 존재하지않음)
  • const a = new set([1,2,3]) : 배열 => set
  • const b = [...a] : set => 배열
  • setName.key(), setName.values(), setName.entries 리턴타입이 setIterator : forEach 순환x
