자바스크립트의 배열은 유동배열 : 크기와 데이터가 변경이 가능하다.

  • const MyArray = ['a','b','c',1,2,3];
  • 데이터의 타입이 다르더라도 허용하고 있음.
  • MyArray = []; 초기값이 없는 빈 배열을 생성 후 데이터를 추가하여 크기 변경이 가능
  • 각 데이터들은 (,)로 분리되고, 자리는 index로 관리 0부터 시작
  • 길이(length) 속성을 이용하여 측정, 크기가 변경될 때 마다 자동으로 체크
  • 인덱스번호 순서 뛰어넘음 등의 변칙적인 사용이 가능함. 모두 undefined(정의되지않음)으로 초기화됨
  • 추후 데이터 통신, 변환에서 자주사용
