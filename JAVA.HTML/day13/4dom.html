DOM(Document obj Model) : 웹페이지에 대한 인터페이스 여러 프로그램들이 페이지의 컨텐츠 및 구조, 그리고 스타일을 읽고, 조작할수 있도록 제공되는 것

  • 엘리먼츠 가져오기11
  • document.getElementById("아이디명") : 리턴 단일 엘리먼츠
  • document.getElementByClassName("ClassName") 리턴타입이 복수 엘리먼트 : HTMLCollection 타입으로 저장
  • document.getElementByTagName("TagName") : 리턴 타입 복수 엘리먼츠 : HTMLCollection 타입으로 저장
  • getElement로 가져오는 객체는 우선적으로 HTMLCollection의 인덱스를 선택
